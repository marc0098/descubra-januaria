# Descubra Januária 🌿

Plataforma turística oficial da cidade de Januária, no norte de Minas Gerais. Explore pontos turísticos, cavernas, guias locais, hospedagem, gastronomia e eventos da região.

[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4-blue.svg)](https://tailwindcss.com/)

## 🏗️ Tecnologias

- **React 19** — Framework base para a UI
- **TypeScript** — Tipagem estática
- **Vite 6** — Build tool ultra-rápido
- **Tailwind CSS 4** — Estilização utilitária
- **Framer Motion** — Animações e transições
- **Lucide React** — Biblioteca de ícones
- **React Router DOM** — Navegação SPA

## 🚀 Como rodar

1. **Instale as dependências:**
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Acesse no navegador:**
   ```
   http://localhost:3000
   ```

## 📦 Build de produção

```bash
npm run build
```
Os arquivos serão gerados na pasta `dist/`.

## 📁 Estrutura

```
src/
├── assets/images/     # Imagens estáticas (hero, logo)
├── components/layout/ # Header, Footer, Layout
├── data/              # Dados JSON (guias, pontos, cavernas)
├── pages/             # Páginas da aplicação
└── styles/            # CSS global e tokens
```

## 🌐 Deploy

O projeto está configurado para deploy na **Vercel** via `vercel.json`.
