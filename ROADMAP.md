# Roadmap de Crescimento - Descubra Januária

Este documento descreve as visões de futuro e possíveis expansões para a plataforma Descubra Januária.

## 🚀 Fase 1: Consolidação (Concluída)
- [x] Reorganização modular da estrutura de pastas.
- [x] Implementação de Path Aliases para desenvolvimento ágil.
- [x] Documentação técnica completa em Português.
- [x] Refatoração de caminhos de ativos e imports.
- [x] Design profissional mobile-first (Guias e Pontos).
- [x] Galeria de fotos interativa nos Pontos Turísticos.

## 📈 Fase 2: Conteúdo Dinâmico (Próximos Passos)
- **Integração com CMS**: Migrar dados estáticos (JSON) para um CMS (Contentful, Sanity ou Strapi).
- **Sistema de Filtros**: Adicionar filtros por categoria em "Hospedagem" e "Eventos".
- **Busca Global**: Um campo de busca que encontra atrativos, eventos e guias.

## 📱 Fase 3: Experiência do Usuário (UX)
- **PWA (Progressive Web App)**: Permitir que o usuário instale o app no celular para acesso offline.
- **Mapas Interativos**: Integração com Google Maps API para localização dos pontos turísticos.
- **Suporte Multi-idioma**: Adicionar suporte a Inglês e Espanhol.

## 🤝 Fase 4: Ecossistema Local
- **Portal do Guia**: Área restrita para guias locais gerenciarem seus perfis e roteiros.
- **Marketplace de Reservas**: Integração com sistemas de reserva de hotéis e restaurantes locais.
- **Cadastro de Guias**: Sistema de cadastro e verificação de novos guias turísticos.
